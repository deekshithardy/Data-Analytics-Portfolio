# SQL Business Insights on Superstore Sales

## 📌 Project Overview
This project analyzes retail sales data using MySQL to generate business insights. The analysis focuses on sales performance, profitability, regional trends, and product categories using SQL queries.

## 🎯 Objectives
- Analyze sales and profit performance.
- Identify top-performing regions, states, and categories.
- Perform data exploration using SQL.
- Apply advanced SQL concepts such as subqueries and window functions.

## 🛠️ Tech Stack
- MySQL 8.0
- MySQL Workbench
- VS Code

## 📂 Dataset
- Sample Superstore Dataset
- Total Records: 9,994

## 📚 SQL Concepts Used
- SELECT
- DISTINCT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- LIMIT
- Aggregate Functions
- Subqueries
- Window Functions (RANK, DENSE_RANK)

## 📊 Business Questions Answered
1. What is the total sales?
2. What is the total profit?
3. Which category generates the highest sales?
4. Which region performs the best?
5. Which states generate the highest revenue?
6. Which sub-categories are most profitable?
7. What is the average sales per transaction?
8. Which transactions exceed the average sales?
9. Which categories have sales above ₹500000?
10. Rank states based on total sales.
